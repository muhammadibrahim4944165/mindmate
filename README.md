
# Mindmate

Mindmate is a modern productivity and wellness app built with Flutter. It helps you organize your life by managing notes, tasks, habits, meetings, and your device calendar—all in one place.

## Features

- **Notes**: Create, edit, and organize notes in a beautiful grid layout inspired by Google Keep.
- **Tasks**: Manage your to-dos with a checklist interface and drag-to-reorder, similar to Google Tasks.
- **Habits**: Track your daily habits with a colorful, streak-based grid inspired by HabitKit.
- **Meetings**: Record meetings and quickly join Zoom or other meeting apps from within Mindmate.
- **Device Calendar**: View and interact with your device's native calendar events.
- **No Premium Restrictions**: All features are fully available—no paywalls, no subscriptions.

## Getting Started

1. **Clone the repository:**
	```sh
	git clone https://github.com/muhammadibrahim4944165/mindmate.git
	cd mindmate
	```
2. **Install dependencies:**
	```sh
	flutter pub get
	```
3. **Run the app:**
	```sh
	flutter run
	```

## Dependencies
- Flutter
- Hive (local storage)
- device_calendar
- url_launcher
- flutter_sound (for meeting recordings)
